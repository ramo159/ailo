# Ailo — Chatbot (React + Vite)

## Quickstart

```bash
npm install
npm run dev
# then open the local server shown in the terminal
```

## Build for production

```bash
npm run build
npm run preview
```

## Customize

- Update colors in `src/main.css` (CSS variables at the top).
- Logo lives at `src/assets/ailo-logo.png`.
- Page structure/components in `src/components/*` and `src/App.jsx`.
- Replace the demo chat logic in `src/components/ChatPreview.jsx` with your API call.
